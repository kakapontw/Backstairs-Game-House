# Backstairs-Game-House

方便大家取得貝克各項資訊及資源(礦鎬)

### 線上版本：
- [GitHub](https://github.com/kakapontw/Backstairs-Game-House)

### 目前包含：
- Facebook、Youtube、Discord等等相關資訊連結
- ~~圖奇開台通知~~ (1.0.1版移除此功能)
- 聊天室資訊
- 提供礦鎬 (自動點擊"光頭點數"額外獎勵)
